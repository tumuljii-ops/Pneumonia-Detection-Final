# Pneumonia Detection using Deep Learning

## Overview

This project detects Pneumonia from Chest X-Ray images using:

- ANN from scratch (NumPy)
- Custom CNN
- MobileNetV2 Transfer Learning

## Dataset

Chest X-Ray Dataset

Classes:
- NORMAL
- PNEUMONIA

## Results

| Model | Accuracy |
|---------|---------|
| ANN | 74.3% |
| Custom CNN | 87% |
| MobileNetV2 | 91% |

## Classification Report

Precision: 0.88

Recall: 0.96

F1 Score: 0.92

## Tech Stack

- Python

- TensorFlow
- MobileNetV2

- Upload Chest X-Ray
